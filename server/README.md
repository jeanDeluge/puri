## 1. DB 비밀번호 환경변수 설정하기

슬랙에 미리 안내드린 DB의 비밀번호를 환경변수에 담아주세요.

```
export PURI_DB_PASSWORD='?'
```

## 2. (필수 아님) 암호화를 위한 crypto 키를 설정해주세요

사전에 슬랙에 미리 안내드린 키 혹은 원하시는 키를 환경변수에 담아주세요

```
export PURI_SECRET='?'
```

## 3. AWS S3 설정

```
export AWS_SECRET_ACCESS_KEY="?"
```

```
export AWS_ACCESS_KEY="?"
```
